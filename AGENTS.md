# AGENTS.md

## Setup commands
- Install deps: `pnpm install`
- Start dev server: `pnpm dev`
- Run tests: `pnpm test`

## Workflow
- Use `pnpm` for all package and script commands
- Run `pnpm test` before opening PRs or finalizing major changes
- Keep commits focused and small when possible

## Code style
- TypeScript strict mode
- Single quotes, no semicolons
- Prefer functional patterns where possible
- Keep code and naming clear and consistent
- Avoid unnecessary comments; document only non-obvious decisions

## Quality checks
- Ensure the project builds and tests pass locally
- Follow existing project conventions before introducing new patterns
